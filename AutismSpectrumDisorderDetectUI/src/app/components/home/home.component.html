What is DeepBrain ?

DeepBrain is a software solution based on neural networks that is capable of detecting the prevelance of Autism Spectrum Disorder using EEG data. This behaviour-independant system is effective than the current diagnosis practice which is based on observering the patient's behaviour.

How to use DeepBrain ?

To use DeepBrain, First need to upload EEG data files as 4 epochs.

  1. BASELINE
  2. START
  3. MIDDLE
  4. END

File names must be participantID_epoch.csv files. Then press Predict button.
